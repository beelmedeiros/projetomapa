# Projeto LOGIX

Este projeto é uma aplicação web simples de mobilidade urbana que demonstra funcionalidades de login, cadastro, navegação e gerenciamento de rotas.

## Funcionalidades principais

- autenticação de usuário.
- exibição de botão Acessar mapa apenas quando o usuário está logado.
- exibição dos dados básicos do usuário e plano selecionado.
- adição de paradas no mapa por endereço pesquisado.
- remoção de paradas com botão de excluir.
- cálculo de rota.
- formulário de cadastro com as informações do usuário usuando a API CEP
- Se o usuário não estiver logado não conseguirá acessar o mapa e nem escolher um plano.
